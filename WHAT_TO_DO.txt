1. Install Termux (F-Droid recommended)
2. Update packages and install Python & Git
3. Clone repo using SSH:
   git clone git@github.com:macyb27/s21-mobile-debug.git
4. Navigate to repo folder:
   cd s21-mobile-debug
5. Run bootstrap if needed:
   ./auto_bootstrap.sh
6. Run debug:
   python3 mobile_debug.py
7. Backup & push changes automatically:
   ./auto_git_backup.sh
